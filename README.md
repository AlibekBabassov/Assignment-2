Banking System (Assignment #2)
------------------------------
Name: Babassov Alibek

Group: It-2503

Summary: This project is a mini banking system developed in Java that demonstrates the use of different data structures.
First, a LinkedList is used to store bank accounts, allowing dynamic addition, searching, and updating of account information such as deposits and withdrawals.
Second, a Stack is implemented to manage transaction history. It follows the LIFO principle, which allows the program to store recent transactions and undo the last operation.
Third, a Queue is used in two parts of the system. One queue manages bill payment requests, and another queue handles account opening requests. Both follow the FIFO principle, ensuring that requests are processed in the order they were received.
Additionally, a fixed-size array is used to store predefined bank accounts, showing the difference between static and dynamic data structures.
Finally, all components are integrated into a menu-driven program with three main sections: Bank, ATM, and Admin. Each section provides specific functionalities such as managing accounts, performing transactions, and processing requests.
